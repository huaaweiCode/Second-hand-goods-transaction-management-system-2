# 二手物品交易管理系统2

#### Description
二手物品交易管理系统2具有两种角色，管理员和用户，实现了以下功能：

- 商品发布
- 商品浏览、购买
- 购物车（批量购买）
- 消息系统
 - 自动消息通知（审核结果、商品变动）
 - 站内私信
- 个人中心
  - 管理员商品审核
  - 消息发送/接收
  - 购买历史
  - 发布历史
  - 收藏夹
  - 个人信息
- 物品搜索
- 物品、消息、收藏夹的分页
- 登录和注册
